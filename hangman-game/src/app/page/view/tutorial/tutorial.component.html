Game instructions

Select a word length, then the game will give you a random word. You need to guess what letters are in the word. If you are correct, then the game will reveal all the instances of the letter in the word. If you are wrong, one of the hanged stickman's body part will be revealed. Make enough wrong choices, and you lose! You can always start a new game, and you can see the instructions again at any time during the game.
